Independent Projects

These independent projects reflects my approach to data analysis, turning complex information into clear insights that drive results. Each project tells a story through data, offering clear, actionable findings that can drive real-world results

  • New York City Waste Collection Trends

    An in-depth analysis of NYC open data to visualize waste and recycling collection trends across the five boroughs.

  • International Trade Sanctions Analysis

    This project involves analyzing a comprehensive international sanctions dataset from 1949 to 2023. I will be using PowerBI, Excel, and SQL to extract key insights and answer specific questions about sanctions trends and impacts.

  • Expense Tracking with Python and Excel

    This project uses Python to process a spreadsheet of expenses in CSV or Excel format. The program automatically:

    • Formats and cleans the data (removing extra symbols, fixing numbers).
    • Organizes expenses by category.
    • Calculates totals for each category.
    • Produces a clean report in either text or CSV format.

    The user receives a clear expense summary showing what was spent, where it was spent, and the total cost per category. The CSV option also provides a structured, ready-to-use file that can be imported into Excel or other tools for further analysis.
